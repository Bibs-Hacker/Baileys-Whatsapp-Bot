## 🌟 Overview

A powerful **WhatsApp Bot** built using **Baileys** that connects directly to your phone and automates messaging, commands, and more.

✨ Features:

* Auto replies 🤖
* Command handling ⚡
* QR-based login 🔐
* Lightweight & fast 🚀

## ⚡ Installation

```bash
git clone https://github.com/bibs-hacker/whatsapp-baileys-bot.git
cd whatsapp-baileys-bot
npm install
```

---

## ▶️ Run the Bot

```bash
node index.js
```

📱 Scan the QR code from your WhatsApp → Linked Devices
